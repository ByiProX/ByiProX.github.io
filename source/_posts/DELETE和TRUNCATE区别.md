|       | DELETE    |TRUNCATE   |
| :-----: | :-----:  | :-----:   |
| 执行速度      | 慢  | 快  |
| 可执行条件    | 可以  | 不可以  |
| 语句分类    | DML  | DDL  |
| 可以回滚事务    | 可以  | 不可以  |
| 删除操作记录日志    | 记录  | 不记录  |
